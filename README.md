##  Proyecto Destacado

### [Coffee Quality Intelligence — Predicción y Recomendación] https://github.com/DataScienceHenryBootCampPF/DataScienceHenryBootCampPF
Sistema integral de Machine Learning para la industria del café de especialidad.

**Lo que incluye:**
- **Modelado Predictivo:** Implementación de modelos (SVR, Gradient Boosting, Random Forest) con un **98.2% de precisión** en la predicción de calidad (Total Cup Points).
- **Sistema de Recomendación:** Motor de recomendación basado en **Clustering K-Means** para segmentar perfiles sensoriales.
- **Infraestructura MLOps:** Pipeline con **MLflow** para tracking de experimentos, **Docker** para contenedores y **GitHub Actions** para CI/CD.
- **Visualización:** Dashboard interactivo en **Power BI** y despliegue de App funcional en **Streamlit Cloud**.

**Tech:** Python · Scikit-learn · MLflow · Docker · Power BI · Streamlit · SQL
